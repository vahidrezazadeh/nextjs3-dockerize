
# DockerFile For Nuxt (3)

Docker File for Nuxt.js (v3) and Node.js (18)



## Usage

Copy Dockerfile and docker-compose.yaml in root project and Run This Command
```
docker-compose up 
```
or This Command to Run in Background
```
docker-compose up -d
```
